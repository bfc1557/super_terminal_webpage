Elementos obrigatórios:
    
    Slider de imagens com tempo definido para trocar (em js);
    Menu de navegação;
    O site deverá apresentar a data e hora, em um lugar definido pelo aluno;
    Um formulário em React com validação das entradas.

    

Pontos a considerar:

    Deve conter toda a matéria lecionada;
    Usar wrappers, ao invês de tabelas;
    O site deve ser flexível, priorizando o uso de %;
    Utilizar as tags semânticas.
